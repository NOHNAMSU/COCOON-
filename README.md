# COCOON Portfolio Site

이 폴더는 디콘 코쿤 앱 소개용 포트폴리오 웹사이트로 제작하였습니다.

- `index.html`을 브라우저에서 열면 바로 볼 수 있습니다.
- `assets/images`에는 최신 `images.zip`의 페이지 이미지 5장을 반영했습니다.
- 메인, 서비스 소개, 기능 설명, 디자인 가이드, 영상 소개는 메뉴 클릭으로 전환되는 페이지 구조입니다.
- 상단 왼쪽 COCOON 로고를 누르면 메인 홈으로 돌아갑니다, 기능 설명에서는 오른쪽으로 구성했습니다.
- 화면 안의 메뉴 위치에는 투명 링크를 얹어 디자인 노출을 해치지 않도록 구성했습니다.
- 메인 페이지에는 첫 진입 페이드, 테라리움 카드 자동 흐름, 도파민 문구 호버 반응, 앱 핵심기능 카드 깊이감 모션을 적용했습니다.
- 서비스 소개 페이지에는 상단 룩북식 이미지 전환, 타깃 카드 호버, 정보구조도 글로우를 적용했습니다.
- 각 페이지 하단의 탑뷰 버튼은 현재 페이지 맨 위로 이동하도록 연결했습니다.
